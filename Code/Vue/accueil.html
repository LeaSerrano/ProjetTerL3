Bienvenue sur ce site destiné à celles et ceux qui souhaitent étudier les Dépendances Fonctionnelles.

La notion de Bases de données et une thématique essentielle de l'Informatique, présente un peu partout dans les site webs : tout type de formulaire, une prise de rendez-vous en ligne, une connexion à un compte... Une base de donnée permet de stocker des informations utiles de façon organisée, pour y avoir accès facilement.

Pour cela, nous travaillons, grâce à des algorithmes, sur la véracité d'un modèle conceptuel comme le modèle Entité- Association ou UML modélisant notre base de données. Ceci nous permet de créer des bases de données avec la certitude que nous ne pouvons pas perdre des informations en utilisant cette dernière. Limitant, la perte ou les fuites d’informations aux seules erreurs humaines des utilisateurs. Outre Entité Association, il existe d'autres approches, plus formelles pour modéliser une base : utiliser les dépendances fonctionnelles. A partir de celles-ci et d'Axiomes d'Armstrong, il est possible de garantir qu'une base est sans redondance et "solide" aux opérations de mises à jour, sans perte d'informations.

Ainsi, dans le cadre de l’Unité d’Enseignement de HAI606I de la faculté des sciences de Montpellier, nous avons développé une application web permettant de simuler le travail sur les dépendances fonctionnelles permettant ainsi l’aide à la conception de base de données sans perte d’information. Cet outil est destiné aux futurs étudiants en informatique et entre notamment en forte corrélation avec l’Unité d’Enseignement HAI502I Système d’information et Base de données 2 proposée par la Facultée des Sciences de Montpellier.
